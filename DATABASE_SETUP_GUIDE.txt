STEP 2: SETUP DATABASE
----------------------
You need a MySQL database from your web hosting provider.

A) Get database credentials from your hosting control panel (cPanel/Plesk):
   - Database Host (usually 'localhost')
   - Database Name
   - Database Username  
   - Database Password

B) Open config.php and update lines 5-8:

define('DB_HOST', 'localhost');              // Usually 'localhost'
define('DB_USER', 'your_database_username'); // From hosting panel
define('DB_PASS', 'your_database_password'); // From hosting panel
define('DB_NAME', 'astrodove_db');           // Database name

C) The database table will be created automatically when you first access api.php

STEP 3: UPLOAD FILES TO YOUR WEB HOSTING
-----------------------------------------
Upload these files to your web server via FTP or File Manager:

Required Files:
- admin-login.html
- admin-dashboard.html
- config.php
- api.php
- All other HTML files (index.html, products.html, etc.)
- Astrodove_logo_transparent.png

STEP 4: UPDATE API ENDPOINT
----------------------------
Open admin-dashboard.html and update the API_URL variable (around line 20):

const API_URL = 'https://yourwebsite.com/api.php';

Replace 'yourwebsite.com' with your actual domain name.

üÜò TROUBLESHOOTING:
===================

Problem: "Database connection failed"
Solution: Check config.php credentials are correct

Problem: Can't login
Solution: Verify username/password in admin-login.html

Problem: Products not showing
Solution: Check API_URL in admin-dashboard.html is correct

Problem: 500 Internal Server Error
Solution: Check PHP error logs on your hosting

Problem: Session expires too quickly
Solution: In admin-login.html, change '24' to more hours (line 134)
